# Magical Heart Particles

A small interactive canvas demo built with p5.js.

## Files

- `interactive_heart.html` — main demo page
- `css/styles.css` — dedicated styles for the demo
- `js/app.js` — animation and interaction logic
- `index.html` — redirect to the demo page

## How to use

1. Open `interactive_heart.html` in a browser.
2. Move your mouse over the canvas to interact with the particles.

## Notes

- The demo uses p5.js from a CDN.
- The animation is responsive and adjusts automatically on resize.
- JavaScript is required to run the interactive effect.
